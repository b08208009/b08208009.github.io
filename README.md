# 在此次實習中，我先觀摩別人如何排版，並在自己的摸索下逐漸改良排版，由上而下逐一做成現在的網站。首先，放上一張地圖背景圖片彰顯自身背景，接著，運用div將文字框與圖片框分開，並追加圖片輪播功能，並將照片的角落變成圓形，使得網頁更生動。
